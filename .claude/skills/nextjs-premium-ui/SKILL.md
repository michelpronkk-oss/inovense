# Skill: nextjs-premium-ui

Use this skill when:
- Building landing pages
- Polishing UI sections
- Improving layout hierarchy
- Creating premium components for Inovense

Objectives:
- Use current Next.js App Router patterns
- Favor Server Components
- Keep UI clean, premium, calm, and responsive
- Use shadcn/ui where it helps consistency
- Use Tailwind classes that are readable and maintainable

Checklist:
1. Inspect existing tokens, spacing, and typography
2. Reuse components before creating new ones
3. Keep sections visually distinct but coherent
4. Ensure mobile polish
5. Avoid unnecessary client-side code
